# Sales Data Analysis (Power BI)

This project analyzes a retail dataset (~2.3M in sales) to evaluate business performance, profitability, and key revenue drivers.

## Key Insights
- Technology and Office Supplies are the main profit drivers
- Furniture underperforms and shows low profitability
- Tables, Bookcases, and Supplies generate consistent losses
- West and East regions outperform others

## Tools Used
- Power BI
- Data Analysis

## Goal
To identify business optimization opportunities and support data-driven decision-making.
